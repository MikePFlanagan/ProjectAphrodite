# Founder Roadmap

This document complements the detailed [product roadmap](./40-ROADMAP.md) and the [Creator Studio roadmap](./roadmap/roadmap.md). It tracks the enduring sequence of product capabilities rather than replacing either plan.

## Now — A Dependable Foundation

- Establish engineering traditions and repository tooling.
- Complete the Entity DNA foundation for persistent character identity.
- Keep the local development environment stable and repeatable.

## Next — Creation and Conversation

- Deliver Creator Studio Alpha.
- Connect identity, appearance, personality, and prompt construction.
- Build reliable live conversation with character-specific responses.

## Then — Memory and Relationship

- Preserve continuity across conversations and model changes.
- Earn relationship progression through shared experience.
- Give users meaningful control over memory and relationship history.

## Later — A Flourishing Ecosystem

- Add safe media, voice, and richer expression.
- Support creators and discovery without compromising trust.
- Build sustainable monetization aligned with user well-being.

At every stage, ask:

> What's the next brick?

And before shipping:

> Does this make the relationship better?
